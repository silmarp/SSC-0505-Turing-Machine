O trabalho foi desenvolvido utilizando a versão do Python 3.9, recomendamos utilizar a mesma versão para evitar problemas.


- Como executar?

Executando pelo python:
1. Digitar o comando 'python main.py'
2. Digitar os inputs na ordem correta

Já que python é uma linguagem interpretada, não geraremos um exe.


- Onde está o código?
O arquivo main.py ficara na raiz da pasta, junto ao manual e relatorio.


- Como compilar?     #não recomendado, por ser uma linguagem interpretada, mas é possivel gerar um binario compilando

1. Por ser uma linguagem interpretada, precisamos apenas garantir a instalação da biblioteca auto-py-to-exe.
2. Comando para instalação: pip install auto-py-to-exe.
3. Depois disso, basta executar o programa.
